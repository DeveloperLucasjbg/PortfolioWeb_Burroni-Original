Metodologìas

Mobile First

Mobile First, como su nombre indica, es una tendencia de diseño y maquetación web orientada a priorizar la visualización en dispositivos móviles.

Responsive Design

El diseño adaptable, es una tendencia de diseño cuyo fin es adaptar las páginas web al dispositivo que se esté utilizando para visitarlas.

SPA

Una single-page application, es una aplicación web que consta de sola página con el fin de dar una mejor experiencia de usuario, como si fuera una aplicación.
